# NCOS Wiki Home

Welcome to the NCOS Wiki. NCOS is an Arch Linux–based distribution built around
the COSMIC desktop environment. Each item below is written as a plain-language
statement so you can find the right page for what you want to do: installing
NCOS, using keyboard shortcuts, adding the NCOS package repository, and keeping
your system up to date.

## Where to Start

These statements tell you which NCOS documentation page answers which question:

- **Download the ISO and create a bootable USB:** The page that covers downloading the NCOS ISO and creating a bootable USB drive is the Pre-Installation Guide (Pre-Install.md).
- **Install NCOS on your computer:** The page that covers installing NCOS from the live environment is the Installation Guide (Installation.md).
- **Learn keyboard shortcuts:** The page that covers opening applications, managing windows, and switching workspaces with the keyboard is the Keyboard Shortcuts page (shortcuts.md).
- **Install NCOS-specific packages:** The page that covers enabling the NCOS package repository for pacman is the Add NCOS Repository page (ncos-repo.md).
- **Update, clean, and maintain the system:** The page that covers updating packages, freeing disk space, managing services, and checking logs is the System Maintenance Guide (update.md).
- **See what changed in the latest release:** The page that covers new features and fixes is the Release Notes page (release-notes.md).
- **Get a general overview of NCOS:** The page that describes what NCOS is and how it is put together is the NCOS Overview page (ncos-overview.md).

## Contributing to NCOS

NCOS is a community-driven project, and contributions are welcome — whether you
are reporting a bug, submitting a patch, or improving this documentation.

- **Contribute to NCOS:** The place to report bugs, submit patches, or improve the documentation is the NCOS GitHub repository at https://github.com/exothermic88/NCOS.

## Getting Help

- **Ask questions about NCOS:** You can ask questions about NCOS directly to the NCOS Assistant applet in the panel, which answers using this documentation.
