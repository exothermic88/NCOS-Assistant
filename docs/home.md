# NCOS Wiki Home

Welcome to the NCOS Wiki. NCOS is an Arch Linux–based distribution built around
the COSMIC desktop environment. This documentation covers installing NCOS, using
keyboard shortcuts, adding the NCOS package repository, and keeping your system
up to date.

## Where to Start

- **Installing NCOS:** Start with the Pre-Installation Guide to download the ISO
  and create a bootable USB, then follow the Installation Guide to install the
  system.
- **Learning the basics:** See the Keyboard Shortcuts page to learn how to open
  applications, manage windows, and switch workspaces.
- **Installing NCOS packages:** See the Add NCOS Repository page to enable
  NCOS-specific packages with pacman.
- **Maintaining your system:** See the System Maintenance Guide for updating,
  cleaning up, and monitoring NCOS.

## Contributing to NCOS

NCOS is a community-driven project, and contributions are welcome — whether you
are reporting a bug, submitting a patch, or improving this documentation. Visit
the NCOS GitHub repository to get involved:
https://github.com/exothermic88/NCOS

## Getting Help

You can ask questions about NCOS directly to the NCOS Assistant applet in the
panel, which answers using this documentation.
